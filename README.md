## 功能介绍

灵境 (SoulLens) 4.0 是一款专业的导演级分镜提示词生成工具，专为 AI 视频生成平台（如 Seedance、可灵、Wan 等）设计。

### 核心功能

- 🎬 **28位世界级导演风格** - 诺兰、维伦纽瓦、斯皮尔伯格、宫崎骏、新海诚等
- 🎥 **智能运镜匹配** - 支持 20+ 种专业运镜技法
- 💡 **光影氛围配置** - 黄金时刻、霓虹 noir、丁达尔效应等
- 🎨 **多视觉风格** - 电影写实、赛博朋克、国风水墨、3D修仙国漫等
- 🛡️ **安全审核预检** - 自动检测并替换敏感词汇
- 📁 **素材管理** - 人物/场景资产库定义
- ⚡ **批量生成** - 一键对比多导演风格
- 🔥 **风格融合** - 智能融合 2-3 位导演特点
- ✏️ **单镜精准修改** - 针对单个镜头进行精细调整
